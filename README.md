# GradlePluginDemo

### 实现功能：<br><br>
1、在打包之前或打包之后执行自定义操作；<br>
2、在打包过程中某个Tash任务之前或之后执行自定义操作；<br>
3、打包的过程中修改字节码；<br><br>
